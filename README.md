## üöÄ Instalasi Lokal

1. Clone repository

   ```bash
   git clone https://github.com/ARVIN1006/Nexa_Digital_Studio.git
   ```

2. Masuk ke direktori project

   ```bash
   cd Nexa_Digital_Studio
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Jalankan server development
   ```bash
   npm run dev
   ```
